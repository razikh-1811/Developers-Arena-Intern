# Task Management API

RESTful Task API built with Node.js, Express, MongoDB, and Mongoose.

## Features
- User registration & login
- JWT authentication
- Task CRUD operations
- User-task relationship
- Pagination & filtering

## Tech Stack
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- JWT

## Setup
1. npm install
2. create .env from .env.example
3. npm run dev
